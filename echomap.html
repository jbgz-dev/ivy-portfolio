Core Experience

A complete loop from listening to spatial memories to completing real-world AI missions.

EchoMap is not a passive map marker system. It connects location context, nearby POIs, user memories, and AI task generation into an actionable exploration experience.

Enter PlaceCreate an exploration session from a real location.
Sense SpaceAnalyze address, weather, nearby POIs, and scene mood.
Hear EchoesBrowse nearby memories left by previous visitors.
Generate MissionTurn a selected echo into a context-aware AI task.
NavigateWalk to a real nearby destination using map routing.
Complete On SiteCapture text, image, or audio at the mission location.
Leave EchoSave a new memory for the next explorer.

AI Architecture

Five AI agents turn map context into executable exploration.

The backend is split into Supabase Edge Function agents. Each agent returns structured output and includes fallback handling for demo stability.

spatial-awareness

Spatial Awareness

Uses reverse geocoding, weather, and nearby POIs to infer scene type, light level, spatial mood, and task hints.

echo-memory

Echo Memory

Retrieves nearby echoes from the database and ranks them by distance, category, and recommendation value.

task-generator

Task Generator

Creates a navigable mission tied to the selected echo, spatial context, and a real nearby POI coordinate.

echo-collect

Echo Collection

Classifies user-submitted text, image, or audio metadata and saves it as a future spatial memory.

result-summary

Result Summary

Summarizes the route, selected echo, completed task, and newly saved memory into a shareable result.
